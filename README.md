# 🎬 Netflix Data Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Netflix Data Analysis Dashboard** built using **Tableau** to explore and visualize Netflix Movies and TV Shows data. The dashboard provides insights into content distribution, ratings, genres, release trends, and country-wise availability, helping users understand Netflix's content landscape through data-driven storytelling.

## 🎯 Objectives

- Analyze Netflix's content library.
- Compare Movies and TV Shows distribution.
- Explore content ratings and genres.
- Identify content growth trends over time.
- Visualize country-wise content contributions.
- Create an interactive dashboard for business insights.

---

## 📊 Dashboard Features

### 1. Content Overview
Displays detailed information about a selected title including:
- Type
- Title
- Rating
- Duration
- Release Year
- Date Added
- Genre
- Description

### 2. Movies vs TV Shows Distribution
Visual representation showing the proportion of Movies and TV Shows available on Netflix.

### 3. Ratings Analysis
Breakdown of Netflix content based on content ratings such as:
- TV-MA
- TV-14
- TV-PG
- R
- PG-13
- NR
- G
- Others

### 4. Genre Analysis
Highlights the most popular genres available on Netflix.

### 5. Country-wise Content Distribution
Interactive world map showing the number of Netflix titles available from different countries.

### 6. Content Growth Trend
Year-wise analysis of Movies and TV Shows added to Netflix.

---

## 🔍 Key Insights

### 📽️ Content Distribution
- Movies account for approximately **68%** of Netflix content.
- TV Shows account for approximately **32%**.

### 📈 Content Growth
- Netflix experienced significant content growth after **2015**.
- The highest content additions occurred between **2017 and 2019**.

### ⭐ Ratings
- **TV-MA** is the most common rating on Netflix.
- Mature audience content dominates the platform.

### 🎭 Genres
Top-performing genres include:
- Documentaries
- Stand-Up Comedy
- International Movies
- Dramas
- Children & Family Movies

### 🌍 Geographic Distribution
- The United States contributes the largest number of titles.
- India is among the top content-producing countries.
